# Change Log Template

Use this template for each deployment or site update.

```md
# <YYYY-MM-DD> - <Short change title>

## Context

What was requested or what problem was found.

## Changes

What changed in the site, code, data, or deployment.

## Checks

Commands or manual checks performed.

## Deployment

Preview/production URL and Vercel project context.

## Risk

Known risks, missing data, or follow-up review needed.

## Rollback

How to undo or revert this change.
```
