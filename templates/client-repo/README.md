# Client Repository Template

Template base para repositorios de clientes criados pela skill `intake-to-repo`.

## Areas principais

1. `site/`: codigo do site.
2. `client-data/`: dados, conteudo e assets do cliente.
3. `logs/`: historico humano de alteracoes feitas com IA.
4. `docs/`: instrucoes operacionais do cliente.
5. `.github/workflows/`: automacoes de CI e review semanal.
